## Como usar
```xml
<!-- Adicionar ao pom.xml -->
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

```xml
<!-- Adicionar a depedency -->
<dependency>
    <groupId>com.github.intopays</groupId>
    <artifactId>intopays-java-sdk</artifactId>
    <version>Tag</version>
</dependency>
```
